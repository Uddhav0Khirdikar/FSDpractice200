# FSDpractice200

Academic submission repository for Full Stack Development (FSD) practice.

This repository contains **200 FSD projects** prepared for learning, practice,
and submission. The collection covers frontend fundamentals, JavaScript
projects, API-based applications, accessibility, performance, dashboards,
games, and full-stack application development.

## Project Summary

| Category | Count | Description |
|---|---:|---|
| Frontend and JavaScript practice projects | 150 | UI projects, API integrations, games, tools, dashboards, accessibility tasks, and performance-focused exercises |
| Full-stack projects with frontend + backend | 50 | Complete project ideas with application UI, backend/API layer, data handling, and practical business workflows |
| Total | 200 | Complete FSD academic practice collection |

## Repository Structure

| Path | Purpose |
|---|---|
| `001-*` to `136-*` | Numbered frontend, JavaScript, API, game, utility, and UI practice projects |
| `134-Generated Full Stack Projects/` | 50 full-stack projects with both frontend and backend scope |
